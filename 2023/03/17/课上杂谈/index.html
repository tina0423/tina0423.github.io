毛概笔记

发表于2023-03-17|更新于2023-03-17
|阅读量:

课上杂谈

资本是普照的光，其他一切都会染上它的色彩

剥削劳动力是最低级的资本，高级的资本会控制媒体、控制国家、控制一切身边的事物

以前的最高统治是可以被推翻的（王朝统治、亨利二世），但是资本的力量渗透到身边的事物（人的观念），无法被轻易推翻

迷茫的统治阶级

社会主义真正的对手不是中小资本、民营资本（邓小平也通过改革开放证明了）而是垄断资本（飞机生产公司两大家，波音和空客）

新封建主义：由资本生产出来的新的等级制度

中国是“修正主义国家”，美国认为中国可能会颠覆已经建立的在资本社会国家的等级制度

资本是一个过程，永远追求更高利益的过程

资本家只不过是资本关系的人格化

使用价值的增加对社会生产的刺激性最小，刺激性最大的是资本（生产100斤-》1000斤的粮食还会腐烂，但把它转换成货币可以作为一种抽象的概念保存、传递）

虚拟是感官世界；符号生产代替了商品生产，属于二次剥削的概念（再掏走你的工资）

不要过度地沉浸在感官世界，容易被资本的逻辑进行二次剥削

技术是一种资本，不能崇拜并且希望技术的进步可以带来劳动力的解放

chatgpt会为资本带来垄断性的利润

社会主义为什么理论出现于西方，但是现实诞生于东方？

西方不出现革命的原因

在西方（英国）生产力发达的时刻，英国选择了继续殖民扩张，所带来的好处反馈给英国的国民，于是英国的工人阶级走上了“投机”主义的道路，与英国资本主义合作。工人阶级已经“贵族化”了，资本阶级吃肉，工人阶级也能分到汤。工人阶级被瓜分在民族之内（特朗普宣扬民族主义，说中国工人抢了美国工人的饭碗），资本主义却全球化了，导致工人阶级难以联合。

东方生产力落后却仍然建立社会主义

革命的主观条件转移到东方，由资本主义全球化，西方必然会殖民东方，从而把当地的 矛盾 推到顶峰，走向革命（俄国和中国）。

因此西方主义有社会主义的客观条件（生产力发达），但是东方有社会主义的主观条件（革命主观性）

但是果实结在东方会造成一系列矛盾；

硅谷银行破产逻辑

新冠 + 金融危机 -> 美联储量化宽松 + 大幅降息 -> 市场主体大量活期存钱至硅谷银行 -> 硅谷银行巨额放贷 + 购买MBS资产抵押债券 和国债对冲低利率 -> 美联储大幅加息 -> (长期)国债和MBS价格下降导致亏损 + 企业由于利率上升从银行取钱还款 -> 硅谷银行抛售债券应对现金流缺乏

新民主主义革命理论

发达资本主义国家利用国债不断救济私人资本

金融危机总会有机构破产，可能是国家也可能是个人企业

新民主主义革命 = “民主革命” + “新”

  • 旧民主革命：资产阶级革封建王朝的命（法国大革命、辛亥革命）
  • 新民主革命：工人阶级领导的革命，革命对象包括封建主义，又包括资本主义和帝国主义

“中国有革命而无变革”，之前的革命只是王朝更替，但是制度没有改变

因此中国新民主主义革命区别于中国以往革命的区别在于改变了统治阶级，将中国千年的弊习扫荡一空

近代中国为什么会发生革命

  • 中华民族面临灭亡的危机

近代中国的国情和社会性质

从独立的、封建的中国变成半殖民地、半封建的中国

中国近代为什么落后了

  • “李约瑟”之问：为什么科学和工业革命没有在近代中国发生？

    中国以特有的生产方式（农业，土地国有）长期没有改变；但是政治却在不断发展

  • 土地国有：土地都归于皇帝，民间地主都是非法的，因为中国会根据土地上的人口征收土地税，若土地被归为私人，则会少上交很多人丁税

  • 赐室、赐地：东方（中国和印度）没有自由的土地，缺乏征服的力量，因为所有生活所需要的物品都是自给自足的（吃的，农业，用的，手工业），导致状态长期不变

  • 只有中国强行遭到西方资本主义国家的入侵之后，才被迫开始发展生产力

  • 郡县制：由皇帝任命所有的人员，结构非常稳定，不像欧洲由于封建制长期处于割裂内斗状态，而打仗缺钱后，就需要向商人借钱，因此商人的地位往往很高，商人和国家紧密结合在一起；相反，中国有超稳定结构（儒家），因此需要重农抑商。

  • 儒家创造出了以一种人际关系为主的社会状态，超越了中国以往原子式的社会状态

  • 欧洲为什么没法统一成一个帝国

    拿破仑、希特勒曾经想要统一帝国，但失败了

    欧盟决策机制：22个国家，所有国家都必须同意才可以实施政策

  • 中国现在：以马克思为本，以传统文化为辅

《大国的兴衰》

官僚资本主义如何产生

  • 在蒋介石打败了北洋军阀后，并没有对地主阶级进行革命，反而是背靠这些旧军阀割据的基础上

    “讨共”蒋介石不愿派出嫡系部队，派周边军阀进行讨伐，但周边军阀也不愿自己的势力受损，两败俱伤、渔翁得利，于是也没有认真打

  • 蒋介石的金融体系：

    • 抗战初期建立的中央银行、中国银行、交通银行、农民银行，由蒋介石亲自任总理事会主席
    • 有蒋家、宋家、孔家、陈家，同时四大家族还进行联姻
    • 金圆券：将百姓的财物以金圆券的方式集中到自己手中，换为了一堆废纸

革命辩证法——蒋介石形式上统一中国的意义

  • 在以阶级对抗为基础的社会形态中，只有当统治阶级已在尽可能高的程度上把自己的权力集结起来的时候，才能爆发伟大的社会变革
  • 蒋介石统一中国军阀是中国共产党革命成功的前提，他将中国所有反革命集团笼在他一人之下，反而为中国毛泽东统一中国奠定了基础，当事人却不知道
  • 毛泽东将国民党分为两派，左派（宋庆龄，宣扬新三民主义，要联合俄、共），右派（蒋介石）；而右派又分为顽固派（蒋介石）和投降派（汪精卫）
  • 顽固派也是可以争取的，一边打一边拉；严厉打击投降派；拉拢左派

汪精卫和投降派

  • 被殖民自由主义的思潮所俘获，认为西方的殖民行为为当地带来了文明，提高了生产力，但实际上带给国家的好处并没有被作用于国民，而是被大国捞取了利益
  • 毛泽东称其为民族投降主义
  • ”大东亚共荣圈“
  • 香港暴乱的青年背着英国的旗帜也是受到这种主义的影响
  • 先发达的国家为后来的国家创造了意识形态的枷锁

新民主主义革命的动力

  • 工人阶级：领导力量
  • 农民阶级：坚固的同盟军
  • 小资产阶级：可靠的同盟军
  • 民族资产阶级：一定时期一定程度的同盟军

中国无产阶级的特殊优点

  • 革命性强：中国无产阶级深受帝国主义、封建主义和资本主义三重压迫，具有强烈的反抗意识
  • 无产阶级和广大农民有天然联系，便于于农民结成亲密的联盟
  • 无产阶级集中在沿海、沿江大中城市，便于组织

中国无产阶级从自发转向自为

  • 共产党人代表着历史运动的利益，代表的是长远的利益。因为他掌握了人类历史运动的规律，并把规律自发的运用到当下的运动中

新民主主义革命的动力——农民阶级

  • 为什么之前没有发现农民是革命的动力？

旧农民的特点

对地主阶级顺从《阿Q正传》

历史主体的建构——灌输论

  • “工人本来也不可能有社会民主主义的意识。这种意识只能从外面灌输进去。各国的历史都证明：工人阶级单靠自己的力量，只能形成工联主义的意识” 工农阶级即没有时间也没有精力，经费去思考这件事情
  • 毛泽东指出政治工作的基本人物是向农民群众灌输这种社会主义思想，批判资本主义倾向
  • 工人阶级由于被压迫的地位，具有社会主义天赋

发展重工业会损害农民利益但会有助于工农总体利益

  • 只有在革命中才能清洗自己身上陈旧的肮脏的东西

革命的目标

  • 政权、族权、神权、夫权

  • 地主政权，是一切权力的基干，除了县长，还有民间地主、土豪劣绅

  • 鼓励了大量的农民参军

用农民的阶级利益动员农民

为什么中国民族资产阶级不能担当领导责任

法国资产阶级/英国资产阶级强大

社会主义改造

第一阶段发展互助组

第二阶段

第三阶段发展高级社

  • 高级社实行生产资料农民集体所有，具有完全的社会主义性质

改革开放以后的新发展：家庭联产承包责任制和土地流转

  • 三权分置：所有权、承包权、经营权

邓小平：农业改革的两个飞跃

“人民公社” -> 家庭联产承包责任制 ->土地流转（为了适应现代化的农业生产）

  • 第一个飞跃：是废除人民公社，实行家庭联产承包责任制
  • 第二个飞跃：适应科学农业发展

即使是家庭联产承包责任制也不能够动摇土地的集体所有制

马克思论美国的“土地平分”

即使每个人都在刚开始拥有相同大小的土地，很快就会有人称为富农，而有人会失去土地，根据土地的肥沃条件、家庭条件，有人会把土地作为交易物品卖给其他人，而一旦卖掉便很难再取回来

毛主席在打土豪分田地后消灭了中国旧地主阶级，但没过两三年就发现出现了新的地主阶级，为了阻止土地的自然分化，推广合作社来把土地国有化集中起来，邓小平为了推动生产力又变回联产连包责任制，而现在中国又开始推动“乡村振兴”

合作社可以看作“集体资本”，每个村民都是股东，还有分红收入

从农民变成流民变成城市资本廉价劳动力，付出的是一个阶级的代价（从有生产资料的农村阶级到无产阶级）

党的有组织》资本的有组织》小农的无组织

党不组织农民，资本就会组织农民

《烟台记事》

  • 价格战发生在生产主体分散
  • 中国为什么没有高端水果：价格战迫使果农放弃水果品质
  • 土地没法规模化经营
  • 中级合作社：生产是一家一户生产，生产资料采购和销售是统一采购（生产领域不合作，流通领域合作）
  • 供销社：统购统销

民族资本阶级

  • 和平赎买（为了防止政局的动乱）

    • 四马分肥（国家所得税，企业公积金，职工福利费，资方红利）
    • 定息制度（核定私股24亿元）

    赎买代表本来这个东西就是自己的

  • 毛泽东早期是允许城乡资本主义的发展，只不过后期收到“全部国有化”的思想影响太深

邓小平理论的精髓

他看到了资本逻辑在当下中国的应有的价值

从私有制到消灭寄生性私有制——寄生性指帝国主义在中国开办的工厂，封建主义的遗留产业等

地摊之所以存在是因为找不到社会化的企业来吸收他们

  • 资本主义私有制的进步作用：

    • 追求货币的劳动激发普遍的勤劳（中世纪的闲散懒惰与游勇斗狠相互补充）
    • 绝对剩余价值生产充分利用劳动者的生产时间（996的正面作用）
    • 相对剩余价值生产使科学技术获得商业价值
    • 资本积累推动流动和信用革命，推动经济全球化进程，使人与人之间的关系充分社会化

  • 生产性私有制与寄生性私有制的界限——平均利润还是垄断利润

  • 当下要防止私有资本的寄生性积累

  • 蚂蚁集团的上市&被叫停：从银行套取贷款，却让全社会承担风险

  • 花呗一年不还所欠的钱与高利贷差不多

  • 要为资本设置红绿灯
