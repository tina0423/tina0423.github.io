2575.找出字符串的可整除数组

给你一个下标从 0 开始的字符串 word ，长度为 n ，由从 0 到 9 的数字组成。另给你一个正整数 m 。

word 的 可整除数组 div 是一个长度为 n 的整数数组，并满足：

  • 如果 word[0,...,i] 所表示的 数值 能被 m 整除，div[i] = 1
  • 否则，div[i] = 0

返回 word 的可整除数组。

示例 2：

提示：

  • 1 <= n <= 105
  • word.length == n
  • word 由数字 0 到 9 组成
  • 1 <= m <= 109

解答：

思路 ：除法判断能不能除尽或者余数其实之于上一次的余数和新增的数有关，以998与9982 分别% 3为例，998 % 3后余2，而9982正是在上者的基础上在‘2’后加上了‘2，即22 % 3，看清之后就可以利用上一位的 partA * 10 + partB来组成新的被除数来看能不能被m除

将10^5位的数字拆分成当前与上一轮的值，变得易于求解
